# Безопасность - Обзор

XiverCRM включает комплексную систему безопасности.

## Основные компоненты

- [Аутентификация](authentication.md) - JWT и API токены
- [Авторизация](authorization.md) - система прав доступа
- [RBAC](rbac.md) - Role-Based Access Control
- [API Tokens](api-tokens.md) - токены для программного доступа

## Рекомендации по безопасности

1. **Измените JWT_SECRET** в production
2. **Используйте HTTPS** в production
3. **Настройте CORS** только для ваших доменов
4. **Регулярно обновляйте** зависимости
5. **Используйте сильные пароли**
6. **Включите audit logs** для мониторинга
7. **Ограничьте API токены** по времени действия
8. **Используйте RBAC** для контроля доступа

## Безопасность паролей

- Минимальная длина: 6 символов (рекомендуется 12+)
- Используйте bcrypt для хеширования
- Не храните пароли в открытом виде

## Безопасность API

- Используйте JWT токены с истечением
- Ограничивайте API токены по времени
- Используйте read-only токены где возможно
- Мониторьте использование токенов

## Безопасность базы данных

- Используйте отдельного пользователя для БД
- Ограничьте права доступа
- Регулярно делайте резервные копии
- Используйте PostgreSQL в production
